# College Event Registration System

A full-stack application for managing college event registrations.

## Features

- Public event browsing and registration
- Admin dashboard for managing events and registrations
- Email notifications
- CSV export for registrations

## Setup

### Backend

1. cd backend
2. npm install
3. Create .env file with MongoDB URI, JWT secret, email credentials
4. npm start

### Frontend

1. cd frontend
2. npm install
3. npm start
